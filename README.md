# Image inpainting frontend app

Image inpainting frontend application made with VueJS framework.

### Requirments

* nodejs >= 10.19.0
* npm >= 6.14.4

### Installing packaged

Run in project root:
```bash
npm install
```

### Running in dev mode

Run in project root:
```bash
$ npm run serve
```

### Building release version

Run in project root:
```bash
npm run build
```
